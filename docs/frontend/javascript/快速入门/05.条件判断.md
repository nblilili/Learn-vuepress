# 条件判断

JavaScript使用`if () { ... } else { ... }`来进行条件判断。

建议永远加上`{}`

```
var age = 20;
if (age >= 18) {
    alert('adult');
} else {
    console.log('age < 18');
    alert('teenager');
}
```

## 多行条件判断

- 如果还要更细致地判断条件，可以使用多个`if...else...`的组合

- 还可以使用 `else...if`
- 注意判断条件的先后顺序

```
var age = 3;
if (age >= 18) {
    alert('adult');
} else if (age >= 6) {
    alert('teenager');
} else {
    alert('kid');
}
```

### 判断条件

JavaScript把`null`、`undefined`、`0`、`NaN`和空字符串`''`视为`false`，其他值一概视为`true`，因此上述代码条件判断的结果是`true`。

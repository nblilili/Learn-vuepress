# 产品概述

视频互动直播将用户分为主播与观众，主播与观众间可进行视频连麦互动，可适用于娱乐社交、游戏直播、在线教育、电商导购等多种视频直播场景。

相较于普通的视频通话，Juphoon 视频互动直播的特点是：

  - 区分主播和观众，主播可以自由发言，其他用户需要通过连麦与主播进行互动。

  - 用户可以随时加入观看直播。

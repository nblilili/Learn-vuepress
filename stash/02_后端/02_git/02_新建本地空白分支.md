title: 02_新建本地空白分支
---

1. pull
2. 新建并切换分支

```
$ git branch 分支名
$ git checkout 分支名

或者执行

$ git checkout --orphan 分支名
```

3. 清理本地分支

```
git rm --cached -r .
git clean -f -d
```

4. 创建空的 commit

```
git commit --allow-empty -m "新分支初始化"
```

5. push

```
git push origin 分支名
```

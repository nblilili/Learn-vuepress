title: 02_导航栏

1. 打开.vuepress/config.js文件，添加下的内容：

```js
// .vuepress/config.js
module.exports = {
  themeConfig: {
    nav: [
      { text: 'Home', link: '/' },
      { text: 'Guide', link: '/guides/' },
      { text: 'External', link: 'https://google.com' },
    ]
  }
}
```

## 分析

- 其实原先的 sidebar 就是平常所见的文章 toc

- 所以我们需要把 sidebar 移到右侧

- 后续对左侧的 sidebar 进行修改

## 步骤

1. 先把 page 页面变小

   修改 `index.styl` 中的`.page` 属性

   ```stylus
   .page
     padding-left $sidebarWidth
     padding-right $sidebarWidth //新增

   ```

2. 复制y一份 `siderbar.vue`，将其命名为 `sidebar-right`

   修改文件中对应的名称为 `sidebar-right`

3. 在 `index.styl` 中复制`.sidebar` 的 css 内容，粘贴后将其命名为 `.sidebar-right,`将其中的 `left` 属性改为 `right`

   ```stylus
   .sidebar-right
     font-size 16px
     background-color #fff
     width $sidebarWidth
     position fixed
     z-index 10
     margin 0
     top $navbarHeight
     right 0 //原为left ==> right
     bottom 0
     box-sizing border-box
     border-right 1px solid $borderColor
     overflow-y auto
   ```

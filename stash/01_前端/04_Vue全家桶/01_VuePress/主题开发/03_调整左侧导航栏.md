## 分析

- 在 vuepress 中没有直接暴露 Vue 对象
- 需要先获取 Vue 对象
- 引入组件包，公司用的是 LayUI，我这边先用bootstrap。方式相同
- 修改左侧导航栏内容

## 步骤

### 添加下拉选

1. 安装 bootstrap 组件库

   ```cmd
   npm install vue bootstrap-vue bootstrap
   ```

2. 引用

   这里需要修改 enhanceApp.js 文件，引入 vuepress 自带的 Vue 对象

   并且添加 bootstrap 组件

   **公司官网用的是 layUI，使用方式同理**

   ```js
   import Vue from 'vue'
   import BootstrapVue from 'bootstrap-vue'

   import 'bootstrap/dist/css/bootstrap.css'
   import 'bootstrap-vue/dist/bootstrap-vue.css'

   // Install BootstrapVue
   Vue.use(BootstrapVue)

   export default ({
       Vue, // VuePress 正在使用的 Vue 构造函数
       options, // 附加到根实例的一些选项
       router, // 当前应用的路由实例
       siteData, // 站点元数据
       isServer // 当前应用配置是处于 服务端渲染 或 客户端
     }) => {
       // ...做一些其他的应用级别的优化
   }
   ```

3. 尝试添加一个下拉选组件

### 复制官网的左边栏

1. 复制了左边的 element 内容

2. 调整为 vue 语法

   ```html
   <!-- links -->
   <div class="left-nav">
     <div class="left-title left-title_1 active" v-for="link in links">
     <a href="">{{ link.name }}</a>
     </div>
   </div>
   ```

3. 修改 css

   ```stylus
   .left-nav
     margin-top: 9px;
     padding: 18px;

   .left-title
     margin: 20px 0;
     cursor: pointer;
     position: relative;
   ```

   官网上其实可以找到 document.css 文件，但是引用的时候出了点问题。//todo 待后续解决

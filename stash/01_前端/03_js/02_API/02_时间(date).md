## 指定时间

创建一个指定的时间对象，需要在构造函数中传递一个表示时间的字符串作为参数

日期的格式 月/日/年 时:分:秒

```js
var d2 = new Date(”12/03/2011 11:10:30“)；
var date = d2.getDate();
```

### getDate()

获取当前日期对象是几号

### getDay()

- 获取当前星期几
- 0 表示周日

### getMonth()

- 获取当前月份

### getFullYear()

- 获取年份

## 时间戳

利用时间戳来测试代码的执行的性能

获取当前的时间戳

```js
time = Date.now();
```
